# Contributing to LogMaster

Thanks for your interest in improving LogMaster!

## Reporting Bugs
- Include log format being analyzed
- Command used and error message
- Sample log lines (sanitized, no sensitive data!)

## Suggesting Features
- Describe the log format or analysis needed
- Use case explanation
- Example log samples

## Pull Requests
1. Fork repository
2. Create feature branch
3. Test with sample logs (never real production logs!)
4. Follow code style
5. Update documentation

## Testing
Test with sanitized, sample log data only. Never use real production logs with sensitive information.
